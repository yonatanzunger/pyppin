pyppin.math.plot_ascii.plot_ascii(data: Callable[[float], float], width: int, height: int, x_axis: Optional[pyppin.math.plot_ascii.Axis] = None, y_axis: Optional[pyppin.math.plot_ascii.Axis] = None, vfill: bool = False) → str¶

Create an ASCII-art plot of some numerical data.

Parameters

  • data – The data to be plotted. If you have a list or a dict, use an Interpolate to turn it into a function.

  • width – The size, in characters, of the output to generate.

  • height – The size, in characters, of the output to generate.

  • x_axis – Options for the axes.

  • y_axis – Options for the axes.

  • vfill – If True, show a vertical fill (in the style of a bar chart) below each value. If False, show the values as points.

Returns

Beautiful ASCII art.
